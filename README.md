# Calidad-Mapa_mental
```mermaid
mindmap
  root[Inspección y Evaluación de Productos]
    (Tipos de Inspección)
      {{Inspección por Atributos}}
        Cualitativas: Se enfoca en características cualitativas
        Clasificación: Los productos se clasifican como defectuosos o no defectuosos
        Ejemplos: color, apariencia, presencia de una etiqueta
      {{Inspección por Variables}}
        Cuantitativas: Se miden características cuantitativas
        Instrumentos: Se utilizan instrumentos de medición para obtener datos numéricos
        Ejemplos: longitud, peso, grosor
    (Métodos de Inspección)
      {{Inspección por Muestreo}}
        Representativa: Se inspecciona una muestra representativa del lote
        Costosa: Se usa cuando la inspección al 100% es costosa o destructiva
        Planes: Existen diferentes planes de muestreo según el nivel de calidad deseado
      {{Inspección en Origen}}
        Producción: Se realiza durante el proceso de producción
        Corrección: Permite identificar y corregir problemas rápidamente
        Ejemplo: inspección de soldaduras en una fábrica
      {{Inspección al 100%}}
        Todos: Se inspeccionan todos los productos
        Critico: Se usa cuando la calidad es crítica o los riesgos son altos
        Ejemplo: inspección de dispositivos médicos
      {{Inspección Final}}
        Envio Previo: Se realiza antes de enviar el producto al cliente
        Requisitos: Asegura que el producto cumple con los requisitos de calidad
        Ejemplo: inspección de automóviles antes de la entrega
    (Técnicas de Inspección)
      {{Inspección Visual}}
        Ojo: Se utiliza el ojo humano para identificar defectos
        Común: Es el método más común y sencillo
        Ejemplo: inspección de la apariencia de un producto
      {{Inspección con Instrumentos de Medición}}
        Herramientas: Se utilizan herramientas como calibradores, micrómetros, etc.
        Preciso: Permite obtener mediciones precisas
        Ejemplo: medición del diámetro de una pieza
      {{Inspección por Pruebas}}
        Simulación: Se simulan las condiciones de uso del producto
        Funcionamiento: Se verifica su funcionamiento y durabilidad
        Ejemplo: pruebas de resistencia de un material
    (Evaluación de Productos)
      {{Análisis de Datos}}
        Estadísticas: Se utilizan técnicas estadísticas para analizar los datos de inspección
        Tendencias: Se identifican tendencias, patrones y posibles problemas de calidad
      {{Comparación con Estándares}}
        Requisitos: Se comparan los resultados de la inspección con los estándares de calidad establecidos
        Cumplimiento: Se determina si el producto cumple con los requisitos
      {{Toma de Decisiones}}
        Acciones: Se toman decisiones basadas en los resultados de la evaluación
        Correctivas: Se pueden implementar acciones correctivas o preventivas
    (Desviación Estándar)
      {{Definición: Medida de dispersión que indica cuánto se alejan los datos individuales de la media}}
      {{Fórmula:    ddfdfdfdfdfdfdblablablablablasdgsdgssdsdsdsdsd}}
        σ: Desviación estándar
        xi: Cada dato individual
        μ: Media de los datos
        N: Número de datos
      {{Uso:}}
        Variabilidad: Evaluar la variabilidad de un proceso o producto
        Comparación: Comparar la variabilidad de diferentes procesos o productos
        Límites: Establecer límites de control para la calidad
        Decisiones: Tomar decisiones basadas en datos
